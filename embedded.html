Certfid (NFC Verification)

2023 NFC Technology
Planning Phase

NFC-based certificate verification system for authenticating physical documents and credentials using embedded NFC tags.

Project goals:

  • Create a system for embedding NFC tags in certificates and credentials
  • Develop secure verification protocol with cryptographic signatures
  • Build mobile app for scanning and verifying NFC tags
  • Implement tamper-evident physical integration
  • Create management system for issuing and revoking credentials

Technologies: NFC, RFID, Microcontrollers, Cryptography

Arduino Rhythm Game Controller

2021 Game Hardware
Completed

Custom rhythm game controller built with Arduino, featuring low-latency input processing and RGB lighting effects.

Controller features:

  • Arduino Pro Micro as the main controller
  • Custom PCB design for button connections

  • USB HID interface for plug-and-play compatibility
